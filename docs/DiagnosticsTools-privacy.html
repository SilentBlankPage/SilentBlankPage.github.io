隐私政策 / Privacy Policy

生效日期（Effective Date）：2026-05-06

1. 概述 / Overview

我们重视你的隐私。DiagnosticsTools 是一款面向技术支持、运维和现场排障的本地诊断分析工具。

We value your privacy. DiagnosticsTools is a local diagnostic analysis app for support, operations, and field troubleshooting.

2. 我们处理的信息 / Information We Process

我们不会主动收集你的个人信息，也不会默认将你在应用中创建的内容上传到开发者服务器。

We do not actively collect your personal information, and we do not upload the content you create in the app to developer-controlled servers by default.

  • 导入的 ZIP 诊断包
  • Imported ZIP diagnostic bundles
  • 解压后的会话文件
  • Extracted session files
  • 解析后的日志和 JSON 记录
  • Parsed logs and JSON records
  • 关键词规则与知识数据
  • Keyword rules and knowledge data
  • 查看偏好与本地设置
  • Viewer preferences and local settings

3. 联网说明 / Network Access

本应用涉及的联网场景通常仅限于以下情况：

Network activity related to this app is typically limited to the following scenarios:

  • 应用默认不会将诊断包或日志内容上传到开发者控制的服务器。
  • The app does not upload diagnostic bundles or log contents to developer-controlled servers by default.
  • 如果你打开外部支持或政策链接，设备可能会在浏览器中访问对应网站。
  • If you open an external support or policy link, your device may access that website in the browser.
  • App Store 的下载和更新由 Apple 处理。
  • App Store downloads and updates are handled by Apple.

4. 权限使用 / Permissions

  • 仅在你通过文件选择器或系统打开方式导入 ZIP 时使用一次性文件访问。
  • Uses the file picker or system open flow for user-selected ZIP imports only.
  • 不会请求通知、日历或定位权限。
  • Does not request notification, calendar, or location permissions.
  • 正常使用不需要额外的隐私敏感系统权限。
  • No additional privacy-sensitive system permissions are required for normal use.

5. 第三方服务 / Third-Party Services

  • Apple App Store
  • Apple App Store
  • 用于文件导入和文档处理的 Apple 系统服务
  • Apple system services used for file import and document handling

6. 你的控制权 / Your Controls

  • 可在设置中删除已导入会话并清理源数据。
  • Delete imported sessions and clear source data in Settings.
  • 无需账号即可打开或关闭本地查看页面。
  • Open or close local viewer pages without an account.
  • 可随时从设备中移除应用。
  • Remove the app from the device at any time.
