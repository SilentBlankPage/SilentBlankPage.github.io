隐私政策 / Privacy Policy

生效日期（Effective Date）：2026-02-21

1. 我们是谁 / Who We Are

Pure Trace 是一款离线运行的日志分析工具，用于日志检索、筛选与问题定位。

Pure Trace is an offline log analysis app for searching, filtering, and troubleshooting logs.

2. 数据采集与存储 / Data Collection and Storage

  • 本地处理：日志文件在设备本地解析，不上传到开发者服务器。
  • 配置信息：关键词、知识库、解析规则等保存在本地沙盒。
  • iCloud 同步（可选）：仅同步配置数据（不含原始日志文件），通过 Apple iCloud 进行。
  • Local processing: log files are parsed locally and are not uploaded to developer servers.
  • Configurations: keywords, knowledge base, and parsing rules are stored in the local sandbox.
  • Optional iCloud sync: only configuration data (excluding raw logs) is synced via Apple iCloud.

3. 权限说明 / Permissions

  • 文件访问权限：用于读取你主动选择的日志文件。
  • 网络权限：仅用于 Apple 订阅验证及可选 iCloud 同步。
  • File access: used to read log files you explicitly select.
  • Network access: only for Apple subscription validation and optional iCloud sync.

4. 第三方服务 / Third-Party Services

我们不集成第三方广告跟踪或数据分析 SDK。

We do not integrate third-party ad tracking or analytics SDKs.
